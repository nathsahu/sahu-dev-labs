## 🛠️ Project Overview
This website serves as a central hub for **Chetnath (Sahu Dev-Labs)**. It features advanced Linux installation guides, a real-time search system, and a modern neon-themed UI optimized for both desktop and mobile (Termux) users.

### ✨ Key Features:
- **Dynamic Typing:** Professional intro using `Typed.js`.
- **Linux Hub:** Step-by-step documentation for **Ubuntu PRoot** and **Kali NetHunter**.
- **One-Click Copy:** Fast command copying for terminal users.
- **Smart Search:** Instant filtering for tutorials and tools.
- **Dark/Light Mode:** Full theme customization with a single toggle.
- **Responsive Design:** Mobile-first layout with a sleek hamburger menu.
- **Contact Integration:** Functional messaging via Formspree.
